# JSON Formatter & Validator { }

A free, open-source JSON formatter, validator, and minifier. Beautify, validate, and compress JSON data instantly with syntax highlighting and error detection.

🔗 **[Live Demo →](https://ismailzahid1995.github.io/json-formatter/)**

## Features

- ✨ **Format / Beautify** — pretty-print JSON with proper indentation
- ✓ **Validate** — instant validation with detailed error messages
- ⚡ **Minify** — compress JSON and see size savings percentage
- 🎨 **Syntax Highlighting** — color-coded keys, strings, numbers, booleans, null
- 🌲 **Tree View** — collapsible tree visualization of your JSON structure
- 📊 **Stats** — file size, key count, max depth, and root type
- ⌨️ **Live Validation** — validates as you type
- 📋 **One-Click Copy** — copy formatted output instantly
- 🌙 **Dark Premium UI** — clean dark design
- 📱 **Fully Responsive** — works on all devices
- 🔒 **100% Client-Side** — nothing leaves your browser

## Tech Stack

- Pure HTML, CSS, and JavaScript — zero dependencies
- Google Fonts (Inter + JetBrains Mono)
- Hosted on GitHub Pages (free)
